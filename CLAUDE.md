# Instructions for Claude Code

This file is the entry point for Claude Code working in the CorpusScan repo. The full conventions for any AI coding agent live in [AGENTS.md](AGENTS.md) — read it.

## TL;DR for Claude

- This is **CorpusScan**, a hackathon MVP. Ship over polish.
- **Backend**: FastAPI in `/backend`, Python 3.12, managed with `uv`.
- **Frontend**: React + Vite + Tailwind in `/frontend`, generated by Lovable from `docs/lovable-prompt.md`.
- The product itself contains AI agents (Finance, Scripter, Hera). They are **not** you. Their system prompts live in `docs/agent-prompts.md`. Do not confuse the product's agents with the AI coding agent (you).
- **No database, no queue, no auth.** Don't add them. If you feel you need one, re-read [PRD.md](PRD.md).
- All API keys are server-side env vars. **Never put a third-party key in the frontend.**
